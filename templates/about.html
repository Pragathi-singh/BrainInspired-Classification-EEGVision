About EEGVision

EEGVision is a brain-inspired image classification demo built for academic submission.

Architecture
  • Backend: Flask + Python
  • Model: PyTorch (ResNet-18 fine-tuned on CIFAR-10)
  • Frontend: Bootstrap 5 + Chart.js
How to run
  1. Ensure dependencies installed (see README / requirements.txt)
  2. Run python app.py
  3. Open http://127.0.0.1:5000
